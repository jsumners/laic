Class: Namespace

Namespace

{Laic} Namespace

Constructor

new Namespace()

A Namespace is a subclass of Laic that provides a unique container for dependencies. Namespaces support all of the methods of the parent Laic instance.

Source:
  • laic.js, line 20
Example
const laic = new Laic('foo');
laic.foo.register('today', new Date());
console.log(laic.foo.get('today')); // today's date
